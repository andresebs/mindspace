Para poder ejecutar la aplicación "MindSpace" se abre el link, luego se escanea el codigo "Hiro" o dado el caso, 
el codigo del modelo corespondiente cuando la app este en su segunda version.
se selecciona el objeto o mueble que queremos proyectar, iniciaremos con el modelo del armario que ya esta cargado.
luego se bloquea su movimiento para que la proyeccion no dependa solo del codigo hiro
al tener el modelo ya bloqueado, podemos interactura con el, de manera que podemos cambiar su tamaño y rotarlo
para poder ubicarlo donde nos parezca mas favorable.

en futuras versiones tendra mas modelos y objetos para que no solo lo pueda usar una persona natural, si no 
que tambien empresas y profesionales en temas de arquitectura, construccion y afines.
